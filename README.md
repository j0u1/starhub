## StarHub

Простой пет-проект, предназначенный для просмотра звёзд конкретного пользователя. Пока что функционал - только поиск по звёздам, так же есть ограничения по парсу звёзд с GitHub, чтобы это обойти нужно внедрить аунтодификацию.

Создался по причине очень неудобного перехода к странице с моими звёздами на GitHub.

![website preview](https://github.com/user-attachments/assets/0c470279-c102-41bc-b6aa-71f5d07716d9)

Стек: `Nuxt`, `Vue`, `Typescript`, `Tailwind CSS`, `Bun`

![stack](https://skillicons.dev/icons?i=nuxt,vue,ts,tailwind,bun)

## Установка

Зависимости

```bash
bun install
```

## Локальный запуск

Для запуска локально по `http://localhost:3000`

```bash
bun run dev
```

## Билд

```bash
bun run build
```
